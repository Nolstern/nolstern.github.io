Tools Used

💻

Cursor

I built the entire site using Cursor, a free AI-powered code editor. It helped me write and organize my HTML, embed SEO tags, and improve structure. Cursor also integrates seamlessly with GitHub, allowing me to upload my code and host the site without needing to buy a domain.

🌐

GitHub Pages

I used GitHub Pages to publish my site for free. It connects with Cursor and makes it easy to deploy updates.

📊

Google Analytics (GA4)

I linked my site to Google Analytics to track site performance and user behavior over time.

🔍

Google Search Console

After completing my content and adding SEO elements like title tags and alt text, I submitted my site to Google Search Console to help it get indexed and monitored.

🤖

Claude

Used Claude to generate design-based code suggestions that I adapted in Cursor. It helped inspire cleaner layouts and improve overall page structure and readability.

Keyword Strategy Breakdown

I focused on choosing relevant keywords that matched what people are actually searching for. These keywords helped shape the direction of my pages and made it easier for search engines to understand what the content was about.

To find strong keyword ideas, I used a combination of free tools and manual research methods.

My keyword research toolkit included:

📊

Ubersuggest

To check search volume, competition, and keyword suggestions

🔍

Google Autocomplete

Start typing a query in the search bar to see real user search suggestions

💡

AnswerThePublic

To explore question-based keyword ideas and topic clusters based on search behavior

Once selected, the keywords were used naturally across the site, including:

Page Titles & Headings

Strategic keyword placement in main page titles and section headings

Meta Descriptions

Compelling descriptions that include target keywords for search results

Body Content

Natural keyword integration throughout main page content

Blog Posts

Focused content targeting specific SEO education keywords

Examples of keywords used:
learn SEO step by step how to start SEO for beginners how long does it take to learn SEO can I teach myself SEO

These keywords supported the overall goal of helping people find useful, beginner-friendly SEO guidance and improving how search engines rank the site.

Skills Learned Through the Project

This project gave me the chance to build and refine practical SEO skills by applying real tools and techniques throughout the entire process. Instead of just reading about concepts, I gained hands-on experience that helped me understand how SEO works in action.

The hands-on approach was key to moving from basic understanding to real-world execution.

Some of the core skills I developed include:

Google Search Console

I learned how to submit a site for indexing, monitor crawl status, and identify how Google views each page.

Google Analytics (GA4)

I set up tracking to evaluate how users interact with the site, measure performance, and interpret key metrics like page views and engagement time.

Keyword Discovery

Using tools like Ubersuggest and Google Autocomplete, I learned how to find relevant keywords and map them to specific content needs.

SEO Content Writing

I practiced embedding keywords naturally in titles, descriptions, and content without affecting readability.

AI-Assisted Development

I used Cursor, Claude, and ChatGPT to write clean HTML, add metadata, and improve layout structure. These AI tools also supported troubleshooting and helped refine formatting and page flow for better clarity and usability.

Internal Linking

I linked pages logically to improve both user navigation and how search engines understand the site structure.

Page Structure & Accessibility

I added alt text, used clear headings, and followed best practices to make pages easy to read and crawl.

This hands-on approach helped me move from basic understanding to real-world execution. I now feel more confident using SEO tools, structuring content for visibility, and managing an entire project from the ground up.
